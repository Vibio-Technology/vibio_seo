# SERP Feature Targeting

Systematically target and win SERP features beyond the blue link.

## 1. Feature Types

| Feature | Click Impact | Best Target For |
|---------|-------------|-----------------|
| Featured Snippet (paragraph) | Position 0, 8-15% CTR | Definitional queries, "what is", "how to" |
| Featured Snippet (list) | High CTR for steps | Process/instruction, top-N lists |
| Featured Snippet (table) | High CTR for comparisons | Specs, pricing, dimensions |
| People Also Ask (PAA) | Mini-ranking per question | Long-tail question keywords |
| Image Pack | 3-5% CTR | Product photos, diagrams, infographics |
| Video Carousel | High engagement | Tutorial, demo, explainer content |
| Knowledge Panel | Brand authority | Branded queries |
| Local Pack | High local CTR | "Near me" queries |
| Sitelinks | 2-6 extra links | Well-structured sites |
| AI Overview | Emerging, cites EEAT sources | Answer-first, original data content |

## 3. Targeting Tactics

### Featured Snippet (Paragraph)
- 40-60 word direct answer after H1 or target H2
- Question as heading → concise answer → expand
- Self-contained, grammatically complete

### Featured Snippet (List)
- `<ol>` or `<ul>` with complete sentences
- Heading matches query intent
- Front-load: Google may truncate at 8 items

### Featured Snippet (Table)
- Real `<table>` with `<th>` headers (not CSS-grid)
- 4-6 columns, 5-10 rows for snippet readability

### People Also Ask
- FAQ section with question H3s
- Answer in 1-3 sentences after each H3
- FAQPage schema

### Image Pack
- Descriptive filenames, keyword-rich alt text
- Image sitemap, WebP format, correct dimensions

### AI Overview Citations
- Answer-first: direct answer in first 100 words
- Original data preferred over aggregated content
- Strong EEAT: clear author, sources, date
- Self-contained paragraphs for passage-level citation
