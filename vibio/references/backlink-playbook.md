# 外链建设手册 — B2B/外贸的权威获取系统

把“外链建设”变成可执行、可核验的权威获取系统：什么时候启动、先建设什么资产、如何筛选真实机会、如何外联、怎样遵守 Google link spam 政策，以及如何用站点自己的结果复盘。品牌提及单独作为 PR/品牌需求/引荐证据追踪，不换算成链接或 AI 排名倍数。

分工声明：`seo-backlinks` 负责外链**分析**（引用域、锚分布、毒链检测、竞品差距数据）；本手册负责外链**获取**（战术、执行、安全）。监控竞品外链动向见 `competitive-war-room.md`。

**执行边界（agent 与人类的分工，全手册适用）**：本手册的所有对外动作——目录注册、账号操作、邮件发送、引用平台响应——由 agent **准备**（筛选目标、草拟邮件/档案文案、整理清单、写入 tracker），由**人类负责人执行发送与注册**。agent 不得尝试自动登录第三方网站填表、不得代表企业发送真实邮件或回复记者请求（除非用户明确接入了发送工具并授权）。outreach tracker 中 agent 只能把状态推进到待人工执行；`sent` 及之后的状态由人类操作后回填。对外沟通涉及企业身份与公开陈述，最终按钮永远在人手里。

---

## 一、原则与启动闸门

**原则：**
- **相关性和编辑价值优先于 DR 数值。** 行业垂直来源通常更可能带来目标读者与业务价值，但不使用“一条胜十条”的固定换算。
- **赚（earn）与正当建设（build），不买排名信用。** 广告/赞助链接按 Google 当前政策使用 `rel="sponsored"`（`nofollow` 仍可接受）；用于操纵排名的买卖、交换和自动化链接属于 link spam。
- **链接是结果，资产是原因。** 没有值得链接的东西，outreach 就是打扰陌生人。

**启动闸门**（这是"不前置外链建设"底线的具体判据）：

| 层级 | 启动条件 | 说明 |
|---|---|---|
| **基础实体/关系链接**（目录/协会/认证/伙伴，见第三节） | 企业真实符合资格、档案能帮助目标市场买家时 | 不以“上线即做完所有目录”为目标；先验证相关性、真实性和维护成本 |
| **针对页面的 outreach** | 页面可索引、能兑现承诺，且存在至少一个真正值得引用的资产 | 不以字数、盲审分数或抽象 phase 代替资产质量和业务价值 |

## 二、链接资产先行（Linkable Assets）

先造值得链的东西，再谈获取。B2B/制造业常见资产类型如下，顺序按企业自身证据、受众和产能决定：

| 资产类型 | 例子（碳纤维厂商） | 为什么赚链 |
|---|---|---|
| **原创数据/行业调查** | 年度复合材料采购价格指数；自家产线 QC 数据统计 | 成为可核验的一手来源，给媒体、工程师和伙伴真实引用理由 |
| **技术参考资源** | 全牌号规格对照表、树脂兼容性速查、CAD/技术图纸库 | 工程师收藏并在论坛/文档中引用 |
| **计算器/选型工具** | 铺层厚度计算器、纤维用量估算器 | 工具类资产年复一年被动获链（复利型） |
| **权威指南（簇支柱页）** | 通过事实、来源、意图和非同质化硬闸门的选型指南 | 资源页收录、教育场景引用 |
| **原创图像/工厂过程影像** | 生产过程实拍、失效模式对比图 | 提供可核验的一手证据和真实引用理由；不把它称为可添加的 E-E-A-T 或排名信号 |

**节奏**：依据真实资产产能、市场事件和历史结果安排；没有研究方法、样本或分发能力时，不为凑年度任务制造“旗舰数据”。

## 三、战术分级表（按 B2B/外贸 ROI 排序）

### Tier 1 — 基础关系与资格机会

| 战术 | 具体清单 | 备注 |
|---|---|---|
| 工业/贸易目录 | 目标买家真实使用、企业有资格、档案有人维护的平台 | 平台名单会变；执行前核验受众、资格、费用、垃圾风险和是否允许官网链接 |
| B2B 电商平台档案 | 企业已经真实运营的 marketplace/supplier profile | 完整准确填写官网字段；不为链接额外创建虚假或无人维护档案 |
| 商会与行业协会 | 企业真实加入的商会/协会会员页 | 价值来自资格、买家发现与关系，不宣称固定“信任权重” |
| 认证机构目录 | ISO 认证方名录、行业认证（航空 NADCAP、汽车 IATF、医疗 FDA 注册等） | 买家用认证目录反查供应商；链接+信任双重价值 |
| 展会参展商页 | 参展/赞助自带的 exhibitor 页链接；讲席带来的会后报道链 | 外贸企业年年参展却常忘了要链接 |
| 客户/供应商/伙伴链接 | 经同意的联合案例、供应商/伙伴资料和真实合作页面 | 以双方读者价值、保密和准确性为前提，不设年度数量 |

### Tier 2 — 运营层：常态化运转的获取渠道

| 战术 | 执行要点 |
|---|---|
| **未链接品牌提及回收** | 定期发现真实相关提及；只有补链能帮助读者时才礼貌请求，并用自身 cohort 记录成功率 |
| **数字 PR（数据资产驱动）** | 用第二节的可核验数据资产向相关媒体/刊物提供独家角度；记录提及、链接、引荐和合格业务结果 |
| **专家引用机会** | 只回应真正符合专业资历的请求；平台、时效和规则在执行前现场核验 |
| **竞品外链差距分析** | 见第四节；“竞品有”只证明线索，不自动证明值得复制 |
| **资源页收录** | 找目标用户真正使用的 resources/useful links 页面，用匹配资产提出编辑建议 |
| **贸易刊物供稿** | 提供原创数据或技术文章，同时衡量真实买家触达和引荐结果 |

### Tier 3 — 高阶层：有余力再做

- **原创研究战役**（大样本调查 + PR 推广），投入大、上限最高。
- **断链建设**：用可用数据找失效的相关被链页，做出真正匹配且更有价值的替代内容再 outreach；成功率由自己的 tracker 建立。
- **大学/技术合作**（.edu）：真实的研究合作/学生项目赞助，不可批量。

### 禁区

PBN、链接交易包、批量垃圾链、目录群发、评论/论坛签名垃圾链、无关行业客座农场、隐藏互惠链接环。判据：如果链接不传递排名价值，它是否仍能帮助真实用户发现相关来源？不能就别做。政策依据以当前 Google link spam 文档为准。

## 四、竞品外链差距分析（可复制链接的最短路径）

1. 选择能代表直接商业、SERP 页型和目标市场的竞品样本；数量由市场和工具覆盖决定。
2. 路由 `seo-backlinks`（免费源）或 `seo-dataforseo`（更全）拉各家引用域。
3. 取交集：链向多个竞品但没链你的域名，可作为相关性和可获得性的候选线索；仍需逐站验证编辑原因和目标用户价值。
4. 按来源类型分组定战术：目录/协会 → 直接注册（Tier 1）；资源页 → 配对资产 outreach；编辑引用 → 数字 PR 角度；客座 → 评估相关性再决定。
5. 可优先审查链向多个真实竞品的域名，但不能称为“几乎确定可复制”；按内容匹配与编辑正当性写入 tracker。
6. 在市场、资产或竞争样本变化时刷新；稳定期不为日历重复抓取。

## 五、Outreach 执行

### 目标资格审查（发信前过一遍，不合格不发）

- 有可核验的真实受众、编辑活动、相关内容和引荐可能性；第三方流量只能标为 provider estimate/proxy，不能写成已确认的一方流量
- 主题与你的行业真实相关
- 有编辑标准（不是"给钱就发"的农场）
- 有真实读者、编辑责任和可核验历史，不是只靠第三方分数伪装的空壳

### 邮件结构（每封都要过个性化检查）

```
主题：具体、与对方内容直接相关（不是 "Collaboration Opportunity"）
1. 称呼真名 + 一句引用对方具体文章/查询的话（证明你真读过）
2. 一句话自我介绍（凭什么开口：身份 + 一手资历）
3. 具体价值：你提供的独家数据/资产/替代资源是什么，为什么对 TA 的读者有用
4. 简单的下一步（"值得补进那篇文章吗？"）+ 联系方式
```

**个性化红线**：必须能说明具体收件人/频道、相关页面/请求，以及你的资产为何帮助对方读者；没有真实关联就不发。跟进次数和间隔遵守对方规则、地区法规和团队历史，尊重拒绝与退订。

### 真实预期

不使用通用回复率、成链率或工时承诺。按渠道、市场、资产类型和个性化方式记录 `prospected -> approved -> sent -> replied -> earned -> referred/qualified`，用自己的 cohort 建立基线。没有历史时把目标标成实验假设，并设置成本/停止条件。

### 写回 tracker

每次发出/跟进/结果都更新 `.vibio/trackers/outreach.md`（字段与 Type 全枚举见 `state-templates.md`：directory / association / certification / trade-show / partner / guest-post / product-review / media-request / digital-PR / expert-quote / reclaim / broken-link / resource）。按执行边界：agent 备好草稿与清单后停在待人工执行，人类发送后回填 `sent` 及后续状态。不记录的 outreach 等于没发生——REVIEW 模式靠它复盘。

## 六、安全与政策

- 不要求发布者使用精确匹配锚，也不人为配比品牌锚/裸 URL/关键词锚。编辑者按上下文自然选择。
- 不以每月引用域数量或固定增长百分比判断安全。检查链接是否来自真实事件、资产和关系，以及是否出现买卖、自动化、站群、隐藏交换等 link spam 模式。
- 不用“稀释锚文本”掩盖人工操纵；停止违规行为并处理能控制的来源。
- Disavow 不是日常清洁工具。仅当存在大量垃圾/人工链接，且已导致或很可能导致 manual action 时考虑，并严格按 Google 当前说明操作：https://support.google.com/webmasters/answer/2648487

## 七、品牌提及与平台观察

真实媒体、伙伴、协会、视频或社区提及可以带来品牌认知、引荐、搜索需求和未来链接机会，但第三方相关性研究不能证明固定 AI 排名权重。

- 记录来源、日期、上下文、是否有链接、引荐访问、合格线索与后续品牌搜索变化。
- 对 AI 平台只做带日期、市场和提示词的可重复观察；不能把单次出现换算成提及价值。
- YouTube/视频只有在真实展示产品、过程、测试或专家证据时才是强资产，不是通用“最强信号”。
- 不制造虚假提及、Wikipedia/Wikidata 条目或社区帖子。
- 平台策略见 `geo-dominance.md`，视频执行见 `video-seo.md`。

## 八、预期管理

- 用自己的相关机会、资产产能、历史 win rate 和业务结果设节奏，不承诺每月固定链接数。
- 抓取、链接发现和搜索效果窗口因页面、站点和查询而异。先验证链接真实存在/可抓取，再按 `.vibio/` 中预定义窗口复盘。
- 谢绝"包上 DR60"式承诺。只能承诺团队可控的资产、研究、审核和已授权 outreach 动作；第三方是否提及或给链属于实测结果，不能预先承诺。

## 九、节奏

| 时机 | 动作 |
|---|---|
| 企业资格/关系发生变化 | 核验相关目录、协会、认证、展会和伙伴档案 |
| 有已批准的 outreach 队列 | 按团队容量发送/跟进并更新 tracker |
| 数据成熟或成本需要决策 | 按渠道/资产 cohort 复盘 won/lost、引荐与合格结果 |
| 市场/竞争变化 | 刷新差距样本与资产机会 |
| 有真实研究能力与分发计划 | 立项原创数据/工具/参考资产 |

## 不要做

- 不买用于操纵排名的链接、不碰 PBN、链接包或自动化 link spam；付费宣传按政策标记并按真实引荐/业务价值评估。
- 不在没有链接资产、页面质量不达标时启动 Tier 2+ outreach——先过第一节闸门。
- 不因抽象 phase 拒绝真实相关的客户、伙伴、协会或媒体引用。
- 不发无真实关联和读者价值的模板邮件。
- 不承诺链接数量兑换排名——能承诺的是产出与动作，不是结果。
- 不例行 disavow，也不因未经证实的“负面 SEO”就提交。
- 不操纵外链锚文本或追求固定比例。
- 不忽略真实品牌提及，但不把它换算成“链接三倍”或 AI 排名分数。
- 不把 outreach 做成无记录的一次性冲刺；节奏由机会质量、合规和团队容量决定。
- agent 不越权执行对外动作——注册、发信、平台响应由人类执行，agent 只备料并停在待人工执行状态。
