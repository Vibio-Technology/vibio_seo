# 外链建设手册 — B2B/外贸的权威获取系统

把"外链建设"从一句模糊的"Month 3 开始做 outreach"变成可执行的系统：什么时候启动、先做哪类链接、去哪找机会、邮件怎么写、什么比例安全、以及 2025-2026 的关键重定位——**品牌提及对 AI 引用的作用已超过纯链接**，外链战术必须同时为两个目标服务。

分工声明：`seo-backlinks` 负责外链**分析**（引用域、锚分布、毒链检测、竞品差距数据）；本手册负责外链**获取**（战术、执行、安全）。监控竞品外链动向见 `competitive-war-room.md`。

**执行边界（agent 与人类的分工，全手册适用）**：本手册的所有对外动作——目录注册、账号操作、邮件发送、引用平台响应——由 agent **准备**（筛选目标、草拟邮件/档案文案、整理清单、写入 tracker），由**人类负责人执行发送与注册**。agent 不得尝试自动登录第三方网站填表、不得代表企业发送真实邮件或回复记者请求（除非用户明确接入了发送工具并授权）。outreach tracker 中 agent 只能把状态推进到待人工执行；`sent` 及之后的状态由人类操作后回填。对外沟通涉及企业身份与公开陈述，最终按钮永远在人手里。

---

## 一、原则与启动闸门

**原则：**
- **相关性 > DR 数值。** 一条行业垂直媒体的链接胜过十条高 DR 泛站链接。
- **赚（earn）与建（build），不买（buy）。** 付费链接必须 `rel="sponsored"`，否则违反 Google 垃圾政策；买链接包 = 花钱买贬值资产（SpamBrain 现在默认直接折价这类链接）。
- **链接是结果，资产是原因。** 没有值得链接的东西，outreach 就是打扰陌生人。

**启动闸门**（这是"不前置外链建设"底线的具体判据）：

| 层级 | 启动条件 | 说明 |
|---|---|---|
| **Tier 1 基础实体链接**（目录/协会/认证/伙伴，见第三节） | 站点上线即可做 | 它们是品牌级/实体级链接，不针对具体排名页，属于"把企业身份登记进行业生态"，任何阶段做都安全且必要 |
| **Tier 2+ 针对页面的 outreach** | 同时满足：① 站内已有 ≥1 个链接资产（第二节）或 ≥5 个深度页（判据：通过 95 分盲审闸门，或 ≥1500 字且含 ≥3 个竞品没有的差异化点）；② 目标词页处于 `authority-cascade.md` Phase 3+ | 与 authority-cascade §6 的时序一致：Phase 1-2 页面靠内容质量和内链就该能排名，外链资源留给竞争区间 |

## 二、链接资产先行（Linkable Assets）

先造值得链的东西，再谈获取。B2B/制造业最能赚链的资产类型，按可行性排序：

| 资产类型 | 例子（碳纤维厂商） | 为什么赚链 |
|---|---|---|
| **原创数据/行业调查** | 年度复合材料采购价格指数；自家产线 QC 数据统计 | 成为一手引用源——媒体、博主、AI 引用都必须指向你 |
| **技术参考资源** | 全牌号规格对照表、树脂兼容性速查、CAD/技术图纸库 | 工程师收藏并在论坛/文档中引用 |
| **计算器/选型工具** | 铺层厚度计算器、纤维用量估算器 | 工具类资产年复一年被动获链（复利型） |
| **权威指南（簇支柱页）** | 已通过 95 分盲审的选型指南 | 资源页收录、教育场景引用 |
| **原创图像/工厂过程影像** | 生产过程实拍、失效模式对比图 | 被引用时天然带来源标注；同时是 EEAT 信号 |

**节奏**：每年至少 1 个旗舰数据资产（调查/指数/基准报告），季度检查已有资产的被链情况（`seo-backlinks`）。

## 三、战术分级表（按 B2B/外贸 ROI 排序）

### Tier 1 — 基础层：先做完，免费、安全、一次性

| 战术 | 具体清单 | 备注 |
|---|---|---|
| 工业/贸易目录 | Thomasnet（北美工业采购必备）、Kompass、GlobalSpec、Europages、MacRAE's、IndustryNet、EC21 | 买家真的在这些平台搜供应商——填完整档案（产品、认证、规格），不是提交个 URL 就完 |
| B2B 电商平台档案 | Alibaba.com / Made-in-China / Global Sources 的认证供应商主页放官网链接 | 外贸厂通常已有店铺，把"company website"字段填上就是零成本的 day-one 链接，常被漏掉 |
| 商会与行业协会 | 本地商会 + 1-2 个行业协会会员页 | 信任信号权重高；会员档案确认带活链；协会的活动页/会员聚焦是二次机会 |
| 认证机构目录 | ISO 认证方名录、行业认证（航空 NADCAP、汽车 IATF、医疗 FDA 注册等） | 买家用认证目录反查供应商；链接+信任双重价值 |
| 展会参展商页 | 参展/赞助自带的 exhibitor 页链接；讲席带来的会后报道链 | 外贸企业年年参展却常忘了要链接 |
| 客户/供应商/伙伴链接 | 互写案例研究（你出案例页，对方"合作供应商/客户案例"页回链）；"trusted supplier" 提及 | 最低成本的编辑型链接；每年挑 2-3 个客户做联合案例 |

### Tier 2 — 运营层：常态化运转的获取渠道

| 战术 | 转化率基准 | 执行要点 |
|---|---|---|
| **未链接品牌提及回收** | 15-40%（全渠道最高） | 每月搜 `"品牌名" -site:自己域名` + Google Alerts；发现提及未链 → 礼貌请求补链（是编辑修改请求，不是新请求，几天内就有结果） |
| **数字 PR（数据资产驱动）** | 从业者评价最高的战术 | 拿第二节的数据资产向行业媒体/贸易刊物递独家角度；提及本身（即使无链）对 AI 引用同样有价值（见第七节） |
| **专家引用平台**（HARO 后时代） | 记者请求响应 10-25% | 现状：Connectively 已关停，Featured.com 收购了 HARO 品牌。并行注册 4-5 个：Qwoted、Featured、Source of Sources（SOS）、Help a B2B Writer、SourceBottle（AU/NZ）；响应速度是第一杠杆——查询发布后 15-60 分钟内回复 |
| **竞品外链差距复制** | 视链接类型 | 见第四节 |
| **资源页收录** | 冷 outreach 基准 | 找 "resources" / "useful links" 页（行业协会、大学课程页、技术博客），用第二节资产去配对 |
| **贸易刊物供稿** | — | 每年向 2-3 家行业刊物递原创数据或技术文章；同时触达真实买家（链接+询盘双收） |

### Tier 3 — 高阶层：有余力再做

- **原创研究战役**（大样本调查 + PR 推广），投入大、上限最高。
- **断链建设**：用 `seo-backlinks` 找竞品的失效被链页（优先近 3-6 个月新断的，竞争小），做出明显更好的替代内容再 outreach；冷转化约 5-8%，替代内容 1:1 对口时可到 ~20%。
- **大学/技术合作**（.edu）：真实的研究合作/学生项目赞助，不可批量。

### 禁区（做了就是给 SpamBrain 送数据）

PBN、链接交易包、Fiverr 批量链、垃圾目录群发、评论区/论坛签名垃圾链、无关行业的客座农场（尤其 AI 批量生成的 guest-post 农场——2025 年 8 月起被垃圾更新点名打击）、隐藏互惠链接环。判据：这条链接如果不传权重你还要吗？不要 → 别做。

## 四、竞品外链差距分析（可复制链接的最短路径）

1. 选 3-5 个竞品：2 个直接商业竞品 + 2 个 SERP 竞品（排你目标词的站）+ 1 个内容强站。
2. 路由 `seo-backlinks`（免费源）或 `seo-dataforseo`（更全）拉各家引用域。
3. 取交集：**链向 ≥2 个竞品但没链你的域名**——既证明了相关性，又证明了可获得性（对方愿意链这个行业）。
4. 按来源类型分组定战术：目录/协会 → 直接注册（Tier 1）；资源页 → 配对资产 outreach；编辑引用 → 数字 PR 角度；客座 → 评估相关性再决定。
5. 优先攻"链向 3+ 竞品"的域名（几乎确定可复制），写入 outreach tracker。
6. 每季度刷新一次差距清单（见第九节节奏）。

## 五、Outreach 执行

### 目标资格审查（发信前过一遍，不合格不发）

- 站点真实有机流量（不是只有 DR 数字的空壳）
- 主题与你的行业真实相关
- 有编辑标准（不是"给钱就发"的农场）
- 引用域 ≥10、不在垃圾名单里（`seo-backlinks` 快查）

### 邮件结构（每封都要过个性化检查）

```
主题：具体、与对方内容直接相关（不是 "Collaboration Opportunity"）
1. 称呼真名 + 一句引用对方具体文章/查询的话（证明你真读过）
2. 一句话自我介绍（凭什么开口：身份 + 一手资历）
3. 具体价值：你提供的独家数据/资产/替代资源是什么，为什么对 TA 的读者有用
4. 简单的下一步（"值得补进那篇文章吗？"）+ 联系方式
```

**个性化红线**：收件人名字、对方具体文章/请求的引用、一个具体的数据点或案例，三者缺一就别发。AI 批量 pitch 泛滥的现在，真实个性化是唯一还在涨的差异化。跟进最多 1-2 次，间隔 4-7 天，之后放弃。

### 真实预期（对客户/自己都按这个口径）

| 渠道 | 回复/转化率 | 备注 |
|---|---|---|
| 冷 outreach（资源页/客座请求） | 回复 ~3-5%，成链更低 | 约 20-30 封冷邮件换 1 条链 |
| 记者/专家引用平台 | 响应成功 10-25% | 速度敏感：15-60 分钟窗口 |
| 断链建设 | 成链 5-8%（对口内容可到 ~20%） | 每条链约 3-5 小时工作量 |
| 未链接提及回收 | 15-40% | 最划算，每月例行 |

### 写回 tracker

每次发出/跟进/结果都更新 `.vibio/trackers/outreach.md`（字段与 Type 全枚举见 `state-templates.md`：directory / association / certification / trade-show / partner / guest-post / product-review / media-request / digital-PR / expert-quote / reclaim / broken-link / resource）。按执行边界：agent 备好草稿与清单后停在待人工执行，人类发送后回填 `sent` 及后续状态。不记录的 outreach 等于没发生——REVIEW 模式靠它复盘。

## 六、锚文本与安全

**外链锚文本分布（与站内规则相反：外链精确匹配要克制）：**

| 锚类型 | 安全占比 | 例 |
|---|---|---|
| 品牌锚 | 30-60% | "Vibio Composites" |
| 裸 URL | 10-15% | "vibio.com" |
| 通用锚 | 10-20% | "this study" / "their data" |
| 部分匹配 | 10-25% | "carbon fiber supplier data from Vibio" |
| **精确匹配** | **≤5-10%** | "carbon fiber fabric" —— >15% 进高风险区 |

**速率安全**：新站/低权重站 2-5 个新引用域/月即正常；提速每 6-8 周不超过 +20-30%；参照竞品的引用域月增速做基准。真正的红旗不是绝对速度，是**模式**：机械均匀的每周等量增长、无内容/PR 事件对应的突增、同足迹站点批量出现（PBN 签名）。

**SpamBrain 现状（2026）**：对低质/过度优化链接的默认处理是**折价**（不传值），不是惩罚；真惩罚留给多信号叠加（坏锚比例 + 异常速率 + 垃圾来源同时出现）。所以过度优化的修法是**稀释而非拆除**——补品牌锚/裸 URL 链把比例拉回来。

**Disavow**：2026 年的定位是窄场景应急工具——只在①收到 unnatural links 手动操作、②确认遭负面 SEO 攻击时使用。日常发现的垃圾链交给算法自动忽略即可；例行 disavow 既无必要还可能误伤。

## 七、品牌提及与 AI 引用（外链战术的 GEO 层）

2025-2026 最重要的重定位：**对 AI 搜索引用而言，品牌提及的相关性（0.664）约为反向链接（0.218）的 3 倍**（Ahrefs 7.5 万品牌研究）；YouTube 提及相关性最高（0.737）。同时链接并未失效——它仍驱动传统排名，而传统排名仍是被 AI 检索到的前置条件。

**战术含义：**
- **无链提及也是资产。** 数字 PR / 贸易刊物 / 播客的品牌名提及，即使没给链接，也在给 ChatGPT/Perplexity/AI Overviews 的引用概率加权。争取链接，但别因为"只给提及不给链"就放弃机会。
- **KPI 双轨**：引用域数量（传统排名）+ 品牌提及量趋势（AI 引用），后者用 Google Alerts / `seo-dataforseo` 的 LLM 提及追踪。
- **YouTube 是被低估的渠道**：第三方评测/访谈/工厂参观视频的提及，对 AI 引用的信号强度超过多数文字链接。
- **平台不重叠**：ChatGPT 与 Perplexity 的引用域仅约 11% 重叠；Perplexity 重 Reddit（其最高引用源）与时效，ChatGPT 重高权威站。多平台铺提及，别单点押注。
- 实体级信号（Wikipedia/Wikidata、Knowledge Panel、sameAs）见 `entity-strategy.md`；AI 平台策略全貌见 `geo-dominance.md`。

## 八、预期管理

- SMB/外贸站**每月 2-5 条相关性高的真实链接就是良好节奏**（口径注意：这里数的是赢得的链接；第六节速率安全数的是新增引用域——同一域名多条链接只算一个引用域，别混用两个口径）。竞争激烈行业的头部页也不过每月 +5-14% 引用域自然增速。
- 链接生效滞后 2-6 个月（爬取 + 权重再计算 + 排名反映），REVIEW 模式复盘时按这个窗口判断，别 4 周没动静就换打法。
- 谢绝"包上 DR60"式承诺——能承诺的是资产产出量、outreach 发送量、赢得链接数，不是排名。

## 不要做

- 不买链接、不碰 PBN、不进链接交易群——SpamBrain 折价 + 多信号叠加才惩罚，但钱一定是白花的。
- 不在没有链接资产、页面质量不达标时启动 Tier 2+ outreach——先过第一节闸门。
- 不给 authority-cascade Phase 1-2 的页面做外链——资源留给 Phase 3+。
- 不发无个性化的模板邮件——名字/具体引用/具体价值三缺一就别发。
- 不承诺链接数量兑换排名——能承诺的是产出与动作，不是结果。
- 不例行 disavow——只在手动操作或确认负面 SEO 时用。
- 不用精确匹配锚堆外链——≤5-10%，品牌锚为主。
- 不忽略无链品牌提及——它对 AI 引用的价值约是链接的 3 倍。
- 不把 outreach 做成一次性冲刺——每周稳定节奏 + tracker 记录，突增突停都是反模式。
- agent 不越权执行对外动作——注册、发信、平台响应由人类执行，agent 只备料并停在待人工执行状态。
